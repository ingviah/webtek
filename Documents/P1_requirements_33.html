Administrative Details

Client: Nonstop FK

Contact: Synne Brovold - team coach

Purpose and goals

The purpose of the site is to unite fans and players while promoting Nonstop FK. The site should contain all the information about the football team, limiting the number of channels for fans and players. It will contain a list of all upcoming and played matches, and information about the players.

Audience

The intended users are players of Nonstop FK as well as the supporters of the team. Both the players and the supporters have a need to follow the team. The players want to get information about when their next activities are, such as practices and matches. The supporters need a platform to follow their favorite team.

Content of the site

The site should contain several sections: “About the team”, “Fixtures”, “Players”, “Results”, “Training”. Each section should contain both written text and images. Each section is a new page, which the user can reach from the homepage.

Functional Requirements

  • The webpage should be able to provide a schedule for upcoming matches.
  • The webpage should provide contact info so the users can make contact if problems occur or if they request more information
  • The webpage should provide link to the Instagram account

Non-Functional Requirements

  • The user should be able to send contact message within a couple of clicks
  • The user should be able to find the next match within a couple of clicks
  • The web page should be loaded within 2 seconds

Final location

The site will not reside after the project is over, so no one will be responsible for the site's maintenance.
